@head @module errno @title errno: Operating system error codes

This module defines symbolic constants that represent different operating system error conditions. The constant names (such as ENOENT) are equivalent to C errno symbolic error names. These constants can be mapped to integer errno values using @ref{CodeToErrno}. @see The error values defined in this module are commonly encountered together with @ref{std::IoError} objects. @end

Functions

@fun CodeToErrno(code as Constant) as Int @desc Return the integer errno matching a symbolic constant from the below list. Return 0 if the errno value is not defined for the current operating system. Example: @example CodeToErrno(ENOENT) -- 2 (exact value may vary across operating systems) @end @end @fun ErrnoToCode(errno as Int) as Constant @desc Return a symbolic constant matching an integer errno value. Raise @ref{ValueError} if no constant is defined for the current operating system, or if the constant is not in the below list. Example: @example ErrnoToCode(2) -- errno::ENOENT @end @end @fun ErrorStr(code as Constant) as Str @desc Return an error message describing a symbolic error code (in English). Return nil if no message is available for the code. Example: @example ErrorStr(ENOENT) -- "No such file or directory" @end @end
