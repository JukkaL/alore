Introduction

Alore is a programming language that supports both dynamic and static typing, and a mixture of both in the same program. This document is an introduction of the Alore type system. It explains how and why to use static types when programming in Alore. @note This document is still work in progress. @end @note The implementation of the Alore type system is still incomplete, but it is already usable for non-trivial tasks. For example, the implementation of the type checker heavily uses static typing. @end
