@head @module math @title math: Mathematical functions

This module contains several basic mathematical functions and a few numeric constants. All operations are implemented using floating point arithmetic, and all functions return Float values. Functions accept integer arguments in addition to floats, but integer arguments are converted to floats internally.

All angles are in radians.

Functions

If x is negative, the magnitude of the result will be chosen to be between Pi / 2 and Pi; otherwise, the magnitude of the result is at most Pi / 2. @end @fun Ceil(x) @desc Return x rounded up to the nearest integer. @end @fun Floor(x) @desc Return x rounded down to the nearest integer. @end @fun Round(x) @desc Return x rounded to the nearest integer. @end @fun Trunc(x) @desc Return x rounded to the nearest integer towards zero. @end @fun Exp(x) @desc Return e**x. @end @fun Log(x) @desc Return the natural logarithm of x.
@note The logarithm in base y can be calculated as Log(x) / Log(y). @end @end @fun IsInf(x) @desc Return a boolean indicating whether x is IEEE 754 positive or negative infinity. @end @fun IsNaN(x) @desc Return a boolean indicating whether x is IEEE 754 NaN (Not-a-Number). @end
