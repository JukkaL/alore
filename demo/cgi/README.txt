CGI examples
------------

These examples use the cgi module. You need a web server with CGI support to
run these (Apache, for example).

hello.alo:
  Trivial CGI 'hello, world' program.
dump.alo:
  Dump request and environment information, useful for debugging CGI problems.
wiki.alo:
  A simple CGI wiki application (check the comments for information on how to
  configure the wiki).
